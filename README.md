# Yumeal
Yumeal is a meal planner and grocery list generator that enables you to select meals for the week and you just have the shopping left to do.
This app is designed with a fitness user persona, meaning that the goal is both to plan meals and to get nutritional information relevant to sport such as calories per day, macro nutrient repartition and so on.

# Why am I doing this ?
The main goal is to learn how to develop in Go with a svelte UI and a docker deployment.
